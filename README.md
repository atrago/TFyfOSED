# 项目简介

## 核心功能
- 前后端代码
- SQL文件
- 万字文档
- 参考导入调试视频
- 讲解视频
- 开发工具安装包
- 协同过滤算法推荐
- 管理员、医生、用户三个角色

## 适用场景
- 卫生健康信息管理
- 用户个性化健康推荐
- 医疗资源分配与优化

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
